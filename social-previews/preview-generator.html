How to Convert SVG to PNG

  1. Open each SVG file in your browser (Chrome/Edge recommended)
  2. Take a screenshot of each preview (Windows: Win + Shift + S)
  3. Crop to 1280x640 pixels
  4. Save as PNG
  5. Upload to GitHub repo settings
Pro Tip: Use https://convertio.co/svg-png/ for quick conversion without installing anything!
