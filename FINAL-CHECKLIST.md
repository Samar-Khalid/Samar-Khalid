### How to Convert SVG to PNG:

**Option 1: Online (Recommended)**
1. Go to https://convertio.co/svg-png/
2. Upload the SVG file
3. Download the PNG
4. Upload to GitHub

**Option 2: Browser Screenshot**
1. Open `preview-generator.html` in browser
2. Take screenshot of each preview
3. Crop to 1280x640 pixels
4. Save as PNG

**Option 3: Python (if you have it)**
```bash
pip install cairosvg
python -c "import cairosvg; cairosvg.svg2png(url='prism-analytics.svg', write_to='prism-analytics.png', output_width=1280, output_height=640)"
```
